Arbitrage Pricing Theory

Overview

    covering expected returns and valuations

    Arbitrage Pricing Theory (APT) is an alternative to CAPM for forecasting expected returns

    not appropriate for consensus expected returns (CAPM) but appropriate for a manager's expected return

    the whole point of Arbitrage is to find an nearly risk free return. APT allows us to obtain this (pg 182)

Issues with CAPM
  • CAPM in theory is the entire market of investmetns, but in practice it is just SP 500, so the CAPM can't exactly be valid
The APT
  • APT states that each stock expected excess return is determined by the stocks factor exposure
  • it postulates a multiple-factor model of excess returns
APT Model
    rn = ∑K Xn,k * bk + un
  • excess returns can be expressed using the following factor model
  • K = number of factors in model
  • Xn,k = exposure of stock n to factor K. Factor loadings.
  • bk = factor returnws for factor k
  • un = stock n specific return. the return that cannot be explained by the factors . idiosyncratic return of the stock
  • correlation of b and u of stock n should be 0
  • correlation of u of stock m and u of stock n should be 0
Expected returns
    fn = E{rn} = ∑K Xn,k * mk
  • mk = factor forcast for factor k
  • expected excess returns on any stock is determined by the stocks factor exposure and the factor forecasts assocaiated with those factors
  • Example
    • the CAPM model is a one factor model
    • K = 1
    • Xn,1 = βn
    • bk = return on benchmark
    • m1 = fM
Portfolio Application to APT
  • amoung and group of N stock there will be an effcient frontier for portfolios made up of the N risky stocks
  • Portfolio Q in the max on efficient frontier
Factor Models Finding qualified models
  • a factor model is qualified if and only if portfolio Q is diversified w.r.t that factor model → among all portfolios w/ same factor exposure as Q, Q has min risk
  • this is commonly thought of as any factor model that captures the important differential modements in the stock will qualify
  • **any factor model that is good at explaining the risk of a divsified portfolio should be nearly qualified as an APT model**
Factor Forecasts
  • forecasting mk

    • Methods

    • simpelest way is to take simple average of historical data of the factor
    • average of past data imply stationarity
    • example of factors
      • 12 month average of finance sector returns
      • 12 average of earnings-to-price
      • bond market beta
Valuation in Theory
